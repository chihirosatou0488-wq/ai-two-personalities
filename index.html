GPT-5.6 ファミリーを深掘り

ひとつの AI ではなく、目的と予算に応じて選べる 3 段階構成。米国政府との調整のもと、当初は限定パートナー向けプレビューから始まりました。

S

GPT-5.6 Sol

フラッグシップ。TerminalBench 2.1 で 88.8%（Sol Ultra は 91.9%）。本格的なコーディングや複雑な調査向け。max reasoning effort と ultra モードに対応。

料金：入力 $5 / 出力 $30（/MTok）

向き：重い開発・調査

T

GPT-5.6 Terra

実用派。前世代 GPT-5.5 と同等以上を、おおよそ半分のコストで。TerminalBench 2.1 は 84.3%。日常業務の自動化に向きます。

料金：入力 $2.5 / 出力 $15（/MTok）

向き：日常の業務自動化

L

GPT-5.6 Luna

いちばん手頃。TerminalBench 2.1 は 82.5% と前世代 GPT-5.5（83.4%）にはわずかに届きませんが、価格の手軽さが持ち味です。

料金：入力 $1 / 出力 $6（/MTok）

向き：軽い会話・下支え

安全面の補足： Sol はサイバー関連でも高い能力を示す一方、OpenAI の Preparedness Framework で「Cyber Critical」閾値には達していないとされています。防御支援を後押ししつつ、悪用を防ぐ多層的な安全対策が組み込まれています。

主な違いを深掘り

段階選択 vs 自律長時間

GPT-5.6 は Sol / Terra / Luna で「どれだけ本気で考えさせるか」を選べます。Fable 5 は細かい指示を待たず、長時間の作業をひとりで進める設計が際立ちます。

Fable の安全ルーティング

サイバー・生物化学・不正な模倣（distillation）などの問い合わせは、自動的に前モデル Opus 4.8 へ振り替え。働くのは全体の 5% 未満とされ、残りは Fable 本来の力を使います。

現場での報告例

Stripe では大規模 Ruby 移行を短時間で終えたとの報告。視覚だけでゲームを進めたり、Mythos 5 では生命科学・サイバー防衛の限定提供でも成果が語られています。

Claude Fable 5 とは

Anthropic が「Mythos級」と呼ぶ最高能力クラスを、一般公開できるよう安全側に調整したモデルです。同じ基盤の制限をゆるめた Mythos 5 は、協力先や防衛・研究用途に限定されています。

ざっくり言うと

コマンドライン系ベンチだけ見ると Sol（特に Ultra）がやや優位に見えます。一方 Fable 5 は数日単位の自律コーディングや、視覚・長期記憶を使う作業で強みを発揮する、という整理がしっくりきます。

どちらを意識すべき？

GPT-5.6 ファミリーが向く人
  • 用途ごとに「本気度」とコストを切り替えたい
  • コーディング・調査でベンチ寄りに強いモデルが欲しい
  • max effort / ultra のような明示的モードが好み
  • Luna〜Sol まで価格帯を幅広く使いたい
Claude Fable 5 が向く人
  • 長時間・大規模なコード移行や自律タスクを任せたい
  • 視覚情報やメモを伴う長い作業が多い
  • Claude.ai / Claude Code / API の流れで使いたい
  • 研究・防衛など限定チャネルの Mythos 系に関心がある
正直なところ

どちらが「勝ち」というより、設計思想が違います。数値で比較しやすいのが GPT-5.6 ファミリー、長く並走して仕事を進める感が強いのが Fable 5、という見方が分かりやすいです。提供範囲もまだ動いている最中なので、実際に触れる環境がある方から試すのが安全です。

よくある質問

GPT-5.6 はもう一般ユーザーも使える？
発表時点では限定プレビューから始まり、提供範囲が段階的に広げられている状態です。お住まいの地域・契約・パートナー経由かどうかでアクセス可否が変わります。
Fable 5 と Mythos 5 の違いは？
同じ Mythos 級の基盤に近い一方、Fable 5 は一般向けに安全側へ調整されています。Mythos 5 は制限をゆるめた版で、防衛・研究などの限定チャネル向けです。料金はどちらも入力 $10 / 出力 $50（100万トークンあたり）とされています。
ベンチマークだけで選んで大丈夫？
TerminalBench はコマンドライン操作の一指標です。長時間の自律作業や視覚・記憶を使うタスクでは並びが変わる可能性があります。用途に近いタスクで試すのがいちばん確実です。
6月中旬に Fable が使えなくなったのはなぜ？
2026年6月12日前後にアクセスが一時停止され、7月1日に世界的に再展開された、という経緯が公開情報としてまとまっています。詳細な原因説明は公式発表の範囲に依存します。
